Standards Governance

CertNode is designed as an open, universal standard for tamper-evident digital receipts. We maintain the reference implementation and coordinate ecosystem development while encouraging independent implementations.

Key Infrastructure

  • JWKS Endpoint: /.well-known/jwks.json
  • Cache Policy: 300 seconds (5 minutes)
  • Rotation Schedule: 90-day maximum key lifetime
  • Algorithm Support: ES256 (ECDSA P-256), EdDSA (Ed25519)

Ecosystem

The CertNode receipt format is implementation-agnostic. Organizations can:

  • Use our hosted service for convenience
  • Implement their own CertNode-compatible service
  • Verify receipts from any CertNode implementation
