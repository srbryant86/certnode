Open Standard + Reference API

CertNode is an open standard for tamper-evident digital receipts. Receipts use RFC 7515 JWS over RFC 8785 JCS with ES256. Any payload or header change invalidates verification under standard JOSE rules.

RFC-Compliant Cryptography

Every CertNode receipt uses RFC 7515 JWS with ECDSA P-256 signatures and RFC 8785 JSON Canonicalization. Built for interoperability - any implementation that follows the standard can generate and verify receipts.
