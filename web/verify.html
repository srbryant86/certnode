Receipt Validator

Paste or drop a receipt and a JWKS to verify authenticity with WebCrypto (ES256/EdDSA over JWS/JCS).

How to Use the Validator

  1. Load your receipt JSON file (contains protected, payload, signature, kid fields)
  2. Load the corresponding JWKS file (contains the public key for verification)
  3. Click "Verify Receipt" to validate the cryptographic signature

Need test data? Use the "Load Sample" buttons below to try with example files.
